# 본인 프로필과 이상형 설문 프로필 수정사항

## 공통

### 외형정보
* 몸무게 삭제
* 체형에 "근육 탄탄" 추가. 보통과 통통 사이에 추가 할 것
* 얼굴상을 해당 내용으로 변경
  - 강아지상
  - 고양이상
  - 사막여우상
  - 공룡상
  - 토끼상
  - 곰상
  - 늑대상
  - 꼬북이상
  - 너구리상
  - 두부상
  - 아랍상
* 눈 옵션을 "유쌍", "무쌍"으로 변경

### 본인&이상형 성향
원하는 성향 추가
* 대화가 잘 통하는 사람
* 자기 일에 열정적인 사람
* 외형적이고 활발한 사람
* 센스있고 유머러스한 사람
* 적극적이고 리드하는 사람
* 배려심 깊고 따뜻한 사람
* 내향적이고 잔잔한 사람
* 즉흥적이고 모험을 즐기는 사람

### 데이트 스타일
* 자연 속에서 힐링 (캠핑/등산/드라이브)
* 활기찬 액티비티 (놀이공화/스포츠/ 여행)
* 편안한 실내 데이트 (집 / 영화 / 보드게임)
* 여유로운 데이트 (카페 / 공원 / 산책)
* 문화생활 즐기기 (전시회 / 공연 / 영화관람)
* 맛집탐방 (음식 / 술 한잔)


### 취미
취미 부분을 "관심사"로 텍스트 교체
* 영화/드라마
* 전시/공연
* 음악/악기
* 스포츠/헬스
* 여행
* 캠핑
* 카페/맛집
* 반려동물
* 패션/스타일
  

### 연애 스타일
* 직진형
* 리드형
* 천천히 진행하는 편
* 리드가 필요한 편
