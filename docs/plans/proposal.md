# 소개퐝 기획서

## 개요
포항에 위치한 대학교 학생들끼리 소개팅을 위한 서비스이다.
포항에 위치한 대학교는 포스텍, 한동대학교, 선린대학교, 포항대학교이다.

어드민이 특정 기간동안만 소개팅 참여 신청을 받는다.
신청시 학교 이메일로 가입하여 포항에 소속된 대학교임을 인증한다.
필요한 인원수가 채워지면 소개팅 신청이 종료된다.
신청한 유저들은 자신의 정보와 자신이 원하는 상대의 정보를 기입한다.
기입한 정보에 가중치를 부여해 가중치 알고리즘으로 '선호도 순위' 생성을 1단계로 한다.
게일-섀플리 알고리즘으로 '매칭 확정'
매치가 확정되면 이메일로 알림을 보낸다. 신청할때 작성한 연락 수단(전화번호, 카카오톡, 인스타그램 등)을 같이 보내서 상대방에게 연락할 수 있도록 한다.

## 서버
서버는 아직 정해지지 않음
상황에 따라 수퍼베이스나 직접 로컬 데이터베이스 운영 고려

## 상세 기능 명세서

### 대시보드
* 왼쪽 상단에 소개퐝 서비스명 기입 그리고 클릭시 대시보드로 이동
* 오른쪽 상단에 로그인 버튼
* 메인 바디에는 서비스 소개하는 한줄과 오른쪽에 사진
* 그 아래에는 소개팅 참여기간과 현채 참여 인원 표시 (예시: 남자 2/10 여자 4/10)
* 소개팅 참여하기 버튼
* 현재 참여 가능한 대학교 로고들 가로로 나열

### 로그인 & 회원가입
* 로그인은 학생 이메일로만 가능하도록 할것
* 구글로그인만 제공

### 회원가입후 
* 처음 회원가입했다면 본인 기본 정보와 원하는 이상형 정보 기입 폼 작성
* 본인정보란
  - 이름 (실명이고 공개되지 않음, UI로 유저에게 명시할것)
  - 닉네임 (서비스에서 사용할 닉네임)
  - 나이 (년도 선택)
  - 학교 (이메일 도메인에 따라 자동 지정)
  - 학과/학부
  - 키 (공개, 비공개 선택 가능)
  - 몸무게 (공개, 비공개 선택 가능)
  - 몸매(마름, 보통, 통통)
  - 얼굴상 (강이지상, 고양이상, 공룡상, 여우상, 너구리상 등등)
  - 눈 (무쌍, 겉쌍, 속쌍등)
  - MBTI (모름 옵션도 줄것)
  - 종교
  - 담배 여부
  - 음주 정도 (예: 주 1회)
  - 타투여부
  - 연락주기 (1-5분, 5-10분, 10-30분, 1시간, 2시간, 3시간, 5시간이상, 하루이상)
  - 취미 (헬스, 스포츠, 독서, 영화/드라마 시청 등등란)

* 이상형 정보란
  - 키
  - 몸무게
  - 몸매
  - 얼굴상
  - 눈
  - MBTI
  - 종교
  - 담배 여부
  - 음주 정도
  - 타투여부
  - 연락주기
  - 취미 

### 소개팅 신청 페이지
* 현재 진행 중인 소개팅 정보 보여줌
* 기간과 모집 인원 그리고 현재 인원 표시
* 신청 버튼
* 안내사항
* 신청버튼을 누르고 기존에 자신이 기입한 프로필과 이상형 프로필을 그대로 쓸건지 물어보고 그대로 신청하거나 수정할수 있도록 함


### 프로필 페이지
* 자신의 프로필과 자신이 기입한 원하는 이상형 정보 보기 가능
* 자신이 신청한 소개팅 정보 표시, 매칭된 상대 정보 확인 가능
* 회원탈퇴 가능

### 어드민 페이지
* 회원 관리
* 소개팅 열기/닫기
* 어드민 기능들


### 매칭 시스템 (서버와 연계)
* 소개팅 참여 기간이 지나기 전에 참여 인원이 다 차면 자동 신청 종료
* 참여 기간이 지났는데도 인원이 부족하면 어드민이 기간을 연장하거나 인원을 줄임
* 신청이 종료되면 자동으로 가중치 알고리즘으로 '선호도 순위' 생성
* 게일-섀플리 알고리즘으로 '매칭 확정'
* 유저에게 이메일로 알림 보내고 프로필 페이지에서도 확인 가능
